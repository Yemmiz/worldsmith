Worldsmith Changelog

• Added generated colour palettes with named swatches and usage roles. • Added palette data to prompts, markdown notes, and implementation JSON. • Pushed the concept card closer to a darker fantasy tool / item-card presentation. • Kept v1.0 layout improvements and v0.95 sanity validation intact. • Added Directed Idea input with Idea Drift slider for seed-based generation. • Added Flora category and Flora follow-up prompt for growth stages and harvest yields. • Added Portrait / Icon Prompt generation for all categories. • Improved armour prompt positioning for body armour, pants, gloves, and boots. • Refined sleeve/robe armour A-pose spacing: arms about 18–22° away from the torso with clear wrist/hip spacing. • Refined texture prompts to avoid decorative inserts and keep textures as clean base materials. • Strengthened texture prompts further so concept/theme words are not interpreted as literal decorative motifs; textures now explicitly request only clean base materials. • Expanded the texture type pool and pushed texture prompts harder toward fewer, bigger repeating shapes for better readability at small sizes. • Added grass and glass texture types, with dedicated material descriptions so those categories produce more suitable surface guidance. • Replaced the separate prompt boxes with tab-style prompt switching, so you can click between the general image prompt, follow-up prompt, and portrait/icon prompt in one compact panel. • Added a dedicated 4–6 frame timeline / animation prompt tab for categories where sequential motion or growth stages make sense. • Refined the timeline prompt so it asks for one coherent continuous action or growth progression, rather than mixing unrelated poses together.
